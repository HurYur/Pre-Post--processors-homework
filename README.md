# [Pre-Post--processors-homework](https://gist.github.com/itspoma/f7c1d866e95f5d5cd7a5784cf75345c6)

## How to Install ##
`$ npm install` install node package manager

`$ npm run build:html` - build HTML<br />
`$ npm run build:css` - build CSS<br />
`$ npm run build` - build HTML and CSS<br />
`$ npm run serve` - run http-server, open page in browser<br />
`$ npm run build:css:autoprefixer` - parse CSS and add vendor prefixes to CSS<br />
`$ npm run deploy` - publish to a gh-pages branch on GitHub<br />

## Як працюємо? ##
  
- клонуєм репозиторій переходимо у branch *develop* і відколюєм branch із назвою "feature/yourTask" наприклад якщо робите task1 частину 1 то бренч у вас повинен називатись "feature/Task1.1". Переходити на створену бренчу і кодите свою частину
- зображення зберігайте у папку **img**
- перед початком своєї частини в html ставимо коментар
- якщо закінчили свою частину, запушайте усі коміти в бренч в якому ви працювали, і переходьте до другої частини(знову потрібно перейти на develop branch і відколотись на "feature/yourpart" наприклад ви вже почали task2 - "feature/Task2"
- відмічайте прогрес на дошці Trello(посилання внизу сторінки)
